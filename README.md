# RICHIO

RICHIO (RICH I/O) is a helper package for the reading and postprocessing of
[RICH](https://gitlab.com/eladtan/RICH/-/tree/master?ref_type=heads), an open
source radiative hydrodynamic simulation code for astrophysics.

## Installation

To install RICHIO, clone this repository and run under the root directory of this repository:

```
pip install -e .
```

and one can import by

```
import richio
```
